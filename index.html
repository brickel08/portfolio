Superhero Sightings

This is my final project for the Software Guild - a full-stack web app built from scratch using Spring framework that records Superhero sightings! I integrated Java code with relational databases using Spring and JDBC Templates to store and retrieve data. The user can add relevant data using various Bootstrap forms. The application features a 'Live Newsfeed' that queries the superhero database and displays the ten most current superhero sightings based on date data. I used Java, Spring Boot, and JDBC Template to query the database to display current data to the user. Not only did I design the backend of the application, but I also designed the front end UI and UX. I hand-drafted multiple wireframes to organize my execution. I used CSS to design the interface in a beautiful and appealing experience. This project challenged me in many different ways, but I was able to create a project that I am very proud of.

Vending Machine

This project is a fully functional vending machine simulator web-page. The web-page communicates to the Software Guild's live web-service to retrieve data, make calculations, and update inventory. I was given wireframes to use as a template to mimic a client's requirements. The layout of the webpage is responsive and uses three different forms to collect data from the user. The user may view all items in the vending machine, add money by clicking buttons, purchase items by clicking selection, and returns correct amount of change with a 'thank you' message.

Guess the Number

A popular guessing game known as "Bulls and Cows." This program generates a random four digit number, where each digit is different, then checks and validates the user's guess against the generated answer. For each round, the user guesses a number and is told the exact and partial digit matches. The program calculates the results, guesses, numbers generated, and stores them in a database. The application uses API calls through Postman to send HTTP requests and receives JSON data.

Ordering System

A console application for a ficticious flooring company ordering system. This project was written using enterprise MVC architecture and follows an agile approach to its development. The app writes/reads data to and from files using standard CRUD operations.
